SOP PENGAJUAN DATA RUTE

A. Ketentuan Pengajuan
  1. Pengajuan dilakukan via SPV pada minggu pertama (tanggal 1–7).
  2. Update dikirim maksimal minggu kedua (tanggal 14).
  3. Pengajuan wajib email ke:
    m.shony@id-tirta.com dan
    benedictus.richard@id-tirta.com
    CC: ASM, KPJ, dan SSD Manager.
  4. Wajib gunakan template resmi RTM.
  5. SLA Pemrosesan 1–3 hari kerja.
B. Aturan Pengisian Data
  1. Kelengkapan: Seluruh sel wajib terisi, pantang kosong.
  2. Jadwal: Isi 1 (kunjungan) atau 0 (tidak ada, termasuk Minggu). Dilarang pakai strip (-) atau huruf O.
  3. ID Sales: Format baku Angka-Strip-Alfanumerik (Cth: 280-R024).
  4. Konsistensi: ID RUTE = ID SALES. Satu nama tidak boleh punya dua ID.
  5. Nama Rute: Harus persis data master.
  6. Outlet: ID PELANGGAN tidak boleh duplikat.
C. Kamus Warna Log
  • Kuning Sel dibiarkan kosong.
  • Merah ID Sales/Rute salah/inkonsisten.
  • Ungu Nama Rute tidak terdaftar.
  • Biru Muda ID Pelanggan duplikat.
  • Oranye Jadwal bukan angka 1 atau 0.
